## 一、用户使用教程

### 1、登录与注册界面

1. 验证用户名：必须由字母，数字下划线组成，并且长度在5-12位
2. 验证密码：必须由字母，数字下划线组成，并且长度为5-12位
3. 验证确认密码：和密码相同
4. 验证码：不能为空，且要正确输入
